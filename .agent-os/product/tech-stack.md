# Technical Stack

- **Framework:** TanStack Start
- **Version:** Latest stable

## JavaScript Framework

- **Framework:** React
- **Router:** TanStack Router

## UI & Styling

- **CSS Framework:** Tailwind CSS (Latest)
- **Component Library:** React Aria
